min and max width

min-width : minimum value paina dhaka pothey ne shrink avudhi

max-width : max value lopla variki shrink avudhi paina avadhu

| Property | Meaning | | ------------ | ------------------------------------------ | | `min-width` | Element **won’t shrink below** this width | | `max-width` | Element **won’t grow beyond** this width | | `min-height` | Element **won’t shrink below** this height | | `max-height` | Element **won’t grow beyond** this height |

1. min-width: if parent 1000px, min-500 isthey , above 500 it will shrink , but bleow it wont shrink. 2. max-width: if parent 1000px, max-500 isthey, 500 lopala variki shrink avudhu cant grow beyond 500. 3. min-height: won't shrink below this height grow above. 4. max-height: above won't shrink, below avudhi =========================== min height / min width:
won't shrink below the specified value , above it will shrink.
max height / max width:
won't shrink above the specified value , will shrink below the specified value

| | ------------ | ------------------------------------------ |

| `max-width` | Element **won’t grow beyond** this width |
